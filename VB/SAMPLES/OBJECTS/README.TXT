This directory contains a sample VB application that uses Oracle's objects features.
This sample application displays PERSON and ADDRESS object instance from  the database.
This sample is based on CUSTOMERS table containing PERSON REF column.Navigating rows in the
CUSTOMER table displays corresponding PERSON and ADDRESS object instance.

You must first follow the following quick instructions before the applications
can be executed. 

1) Make sure that you have created the database alias 'exampledb' in TNSNAMES.ORA.

2) Log into the Oracle database and install the CUSTOMERS table and PERSON,ADDRESS object type
   stored in "objects.sql". This objects.sql can be found in this directory.

4) Start the application.
