This directory contains two sample VB 6.0 applications that retrieves employee
information from the "emp" table using the PL/SQL stored procedures in the EMPLOYEE
package. The VB projects include EMPLOYEE.cls generated by OO4O Code Wizard Add-in.
This file can also be generated using the OO4O Code Wizard command line utility as follows:
oo4ocodewiz scott/tiger@exampledb EMPLOYEE

You must first follow the following quick instructions before the application
can be executed. 

1) Make sure that you have created the database alias 'exampledb' in TNSNAMES.ORA.

2) Install the EMPLOYEE package stored in oraexamp.sql. This file can be found
   in ORACLE_HOME\oo4o dir.

4) Start the application.
