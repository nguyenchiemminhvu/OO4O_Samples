Prerequisites:

	to run the samples you must have:	
	- a database instance, which has scott/tiger account, created
	
To run the samples
	
1) At the dos prompt run:
	svrmgrl,
	connect scott/tiger
	@<path to stock.sql>stock.sql

2) Use IIS to setup StockQuotes as a new virtual dir with read and execute privileges

3) Open SelectCompany.asp in you browser
