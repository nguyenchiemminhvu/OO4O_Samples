Prerequisites:

	to run the samples you must have:	
	- a database instance, which has scott/tiger account, created
	
To run the samples
	
1) At the dos prompt run:
	svrmgrl,
	connect scott/tiger
	@<path to FillDB.sql>FillDB.sql

2) Use IIS to setup ComputerOrderEntry as a new virtual dir with read and execute privileges

3) Open QueryPage.asp from your browser
